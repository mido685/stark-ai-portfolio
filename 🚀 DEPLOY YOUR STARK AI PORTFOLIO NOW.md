## ⚡ Quick Start (5 Minutes)

### Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Name it: `stark-ai-portfolio`
3. Make it **Public**
4. Click **Create repository**

### Step 2: Push Your Code to GitHub

Copy and paste these commands in your terminal:

```bash
cd /home/ubuntu/stark-ai-portfolio

git remote add origin https://github.com/YOUR_USERNAME/stark-ai-portfolio.git

git push -u origin main
```

Replace `YOUR_USERNAME` with your actual GitHub username.

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Select **Deploy from a branch**
4. Choose branch: `main`
5. Choose folder: `/ (root)`
6. Click **Save**

### Step 4: Your Website is Live! 🎉

Visit: `https://YOUR_USERNAME.github.io/stark-ai-portfolio/`

## 🚨 Troubleshooting

### Site not showing up after 5 minutes
- Check GitHub Actions tab for build errors
- Verify GitHub Pages is enabled
- Try refreshing the page

### CSS/JavaScript not loading
- Clear browser cache (Ctrl+Shift+Delete)
- Check file paths in HTML are correct
- Verify all files are committed and pushed

### Dashboard not opening
- Try keyboard shortcut: Ctrl+D
- Check browser console (F12) for errors
- Enable JavaScript in browser settings

### Newsletter not working
- Check browser localStorage is enabled
- Try a different email address
- Clear browser cache and try again
